# Criptolite

App PHP con login, registro, dashboard, recargas y retiros. Base de datos MySQL.

## Poner en marcha

1. **Base de datos**: Copia `db.example.php` como `db.php` y configura host, usuario, contraseña y nombre de la base.
2. **Servidor**: Sube los archivos a un hosting con PHP y MySQL, o usa XAMPP/WAMP en local.
3. Asegúrate de tener la tabla `users` (y las que use la app) creadas en MySQL.
